Amazon Web Services (AWS) is a secure cloud services platform, offering compute power, database storage, content delivery and other functionality to help businesses scale and grow. Explore how millions of customers are currently leveraging AWS cloud products and solutions to build sophisticated applications with increased flexibility, scalability and reliability.

Since the beginning of its existence, Mitoc Group provided enterprise customers with innovative and cutting edge technology, powered by Amazon Web Services (AWS). Our mission is to enable businesses and developers to provide high-value products and services in a constantly changing digital world. We aim to help end users (especially non-technical ones) to benefit from cloud promise of solving business needs through simple and secure technology.

Mitoc Group is the creator of DEEP Framework, the open source full-stack JavaScript framework for building cloud-native web applications. DEEP (also known as Digital Enterprise End-to-end Platform) empowers developers to use a large variety of serverless components from AWS. The typical web application using DEEP Framework consists of the following services:

  • Security tier: Amazon IAM + Amazon Cognito
  • Front-end tier: Amazon S3 + Amazon CloudFront
  • Back-end tier: Amazon API Gateway + AWS Lambda + Amazon SNS
  • Data tier: Amazon DynamoDB + Amazon SQS / Amazon Kinesis + Amazon Elasticache / Amazon Elasticsearch
  • Monitoring tier: Amazon CloudWatch + Amazon CloudTrail + Amazon Elasticsearch
